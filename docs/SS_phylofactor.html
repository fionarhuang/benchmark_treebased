Data simulation

We simulate a scenario (SS) that only two branches on the tree have differential abundance (signal) between groups. The data is simulated by swapping proportions of two branches, and leaves in the same branch have the same fold change.

# generate a random tree
set.seed(1)
n <- 100
tree <- rtree(n)
# generate a random probility vector  for leaves
p <- rbeta(n = n, shape1 = 2, shape2 = 5)
p <- p/sum(p)
names(p) <- tree$tip.label
# # simulate counts by sampling from multinomial distribution
lse <- simMult(pr = p, libSize = 1000, tree = tree,
               minTip.A = 10, maxTip.A = 20,
               ratio = 1.5, scenario = "SS",
               nSam = c(20, 20), pct = 0.6)
